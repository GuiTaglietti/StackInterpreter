Interpretador

Interpretador de pilha - versão web

Ideia do projeto: Trabalho final da disciplina de arquitetura e organização de computadores, ministrada pelo incrível professor, Marcos José Brusso.

A versão desktop é implementada em C++, usando o QTCreator para a criação de uma interface gráfica. Resolvi fazer uma versão web por diversão e facilitar acesso, utilizando HTML, CSS e JavaScript apenas, com intuito de uma "brincadeira" e open source, sendo disponibilizado no meu GitHub.
